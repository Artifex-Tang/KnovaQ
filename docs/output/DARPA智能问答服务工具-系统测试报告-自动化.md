## 4. 认证架构验证

### 4.1 双认证机制

| 认证方式 | 路径 | 验证结果 |
|---------|------|---------|
| API Key | `/api/v1/*` (Bearer ragflow-xxx) | ✅ 7个端点全通过 |
| Session JWT | `/v1/*` (ragflow login token) | ✅ 6个端点全通过 |
| gaisoft Token | `/prod-api/*` (gaisoft login) | ✅ 全套系统管理API通过 |

### 4.2 RSA加密

- gaisoft 数据库存储 RSA 加密密码 (sys_config id=103)
- 加密公钥使用 ragflow 0.18.0 静态 `public.pem`
- GetAuthorization.java 发送加密密码至 ragflow `/v1/user/login`
- ragflow 使用 `private.pem` 解密验证
